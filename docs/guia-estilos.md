# Guía de Estilos y Componentes

## 1. Paleta de Colores
Diseñada para cumplir con los ratios de contraste WCAG (mínimo 4.5:1 para texto).

| Aplicación | Color | Hex Code |
| :--- | :--- | :--- |
| **Principal** | Rosa Suave | `#FFC4C4` |
| **Secundario** | Azul Claro | `#C4DBFF` |
| **Banner Principal** | Crema | `#FFEBC5` |
| **Barras/Imágenes** | Dorado Mate | `#EECA88` |

## 2. Tipografía
Se elige una fuente sin serifa para mejorar la legibilidad en pantallas.
- **Fuente:** Inter
- **Tamaño Base:** 16px (Cuerpo de texto)
- **Tamaño Destacado:** 18px (Descripciones y etiquetas)
- **Line Height:** 1.5 (Para mejorar la lectura en personas con dislexia o baja visión).

## 3. Componentes de UI
### Botones (Buttons)
- **Estado Reposo:** Color `#FFC4C4` con texto oscuro.
- **Estado Focus:** Borde grueso de 3px color azul `#C4DBFF` para identificar la posición del cursor de teclado.

### Tarjetas de Producto (Cards)
- **Contenedor:** Fondo blanco con sombra ligera para dar profundidad.
- **Accesibilidad:** Toda la tarjeta es un área de clic activa (mínimo 44x44px).
